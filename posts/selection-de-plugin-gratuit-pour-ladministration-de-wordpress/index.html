5 plugins gratuits pour l'administration de WordPress

Quand on parle de plugin, on pense la plupart du temps a ajouter de nouvelles fonctions, se faciliter la tache, gagner du temps du coté utilisateur, le coté User first surement. On oublie plus souvent qu’il existe aussi des plugins ayant pour but d’optimiser l’espace qui vous sera le plus familier, l’interface d’administration.

Voici une selection de 5 plugins pour vous, je m’attache a les tester en profondeur, et si jamais vous avez des retours ou des suggestions, je suis preneur !

1 - Adminimize

Adminimize est un plugin WordPress qui vous permet de cacher les fonctions dans l’interface admin. Pratique pour gérer finement ce que peut réaliser la personne connectée sur la partie admin, en fonction de son niveau.

2 - Admin Management Xtended

Admin Management Xtended vous permet d’ajouter des icônes sur les panneaux Articles/Pages qui facilite la gestion de l’affichage, la date de publication et même le titre d’une multitude de pages et/ou d’articles. Tout ce petit monde roule avec Ajax, vous n’aurez plus besoin de raffraîchir la page a chaque modification

3 - Login LockDown

Permet de limiter le nombre de tentatives de login sur votre interface d’admin. Très simple à utiliser, léger et efficace, Login LockDown fait parti des plugins indispensables quand on parle sécurité. Il fait une seule chose mais il le fait bien.

4 - WP-DBManager

Permet d’optimiser, de réparer, de sauvegarder, de restaurer, de supprimer votre base de donnée. On peut programmer des tâches qui s’exécutent. Vous pouvez recevoir un email hebdomadaire avec une sauvegarde de votre base de donnée MySQL, automatiser les optimisations de la base de donnée. Bref, WP-DBManager est un bon outil pour gérer votre BDD.

5 - All in One SEO Pack

All in One SEO Pack optimise Worpdress pour les moteurs de recherche (Search Engine Optimization).
